Alerts

Les alertes permettent d'afficher des messages importants à l'utilisateur avec différents niveaux de priorité.

Types d'alertes :

Information importante ! Ceci est une alerte primaire avec une icône. Succès ! Votre action a été effectuée avec succès. Attention ! Veuillez vérifier vos informations avant de continuer. Erreur ! Une erreur s'est produite lors du traitement de votre demande. Astuce : Vous pouvez utiliser les raccourcis clavier pour naviguer plus rapidement.

Alertes avec liens :

Cette alerte contient un lien utile pour plus d'informations.

Alertes avec titre :

Vous avez terminé toutes les étapes avec succès. Votre compte est maintenant configuré.

Vous pouvez maintenant commencer à utiliser toutes les fonctionnalités.

Alertes fermables :

Notification : Cette alerte peut être fermée en cliquant sur le bouton X. Nouvelle fonctionnalité : Découvrez notre nouveau système de notifications.

Alertes avec icônes personnalisées :

Fonctionnalité premium : Accédez à des fonctionnalités avancées avec notre plan premium. Merci ! Nous apprécions votre fidélité et votre confiance. Téléchargement disponible : Une nouvelle version est disponible pour téléchargement.

Code d'exemple :

<!-- Alerte simple -->
<dt-alert variant="success">
  <strong>Succès !</strong> Votre action a été effectuée.
</dt-alert>

<!-- Alerte avec titre -->
<dt-alert variant="primary" title="Information importante">
  <p>Contenu de l'alerte avec plus de détails.</p>
</dt-alert>

<!-- Alerte fermable -->
<dt-alert variant="warning">
  Cette alerte peut être fermée.
</dt-alert>

<!-- Alerte avec icône personnalisée -->
<dt-alert variant="info" icon="download">
  Téléchargement disponible
</dt-alert>

Tooltips

Les tooltips permettent d'afficher des informations contextuelles au survol ou au focus d'un élément.

Web Component (Recommandé) :

Utilisez le web component <dt-tooltip> pour des tooltips animés :

Bouton avec tooltip Tooltip droite Tooltip bas Tooltip gauche

Thèmes disponibles :

? ? ? ? ? ?

Avec icônes :

Délais personnalisés :

Rapide (200ms) Normal (500ms) Lent (1000ms)

Tooltips avec contenu long :

Tooltip long

Code d'exemple :

<!-- Tooltip simple -->
<dt-tooltip text="Ceci est un tooltip" position="top" theme="dark">
  <dt-button variant="primary">Bouton</dt-button>
</dt-tooltip>

<!-- Tooltip avec icône -->
<dt-tooltip text="Informations" position="right" theme="primary">
  <dt-icon name="info" size="lg"></dt-icon>
</dt-tooltip>

<!-- Tooltip avec délai personnalisé -->
<dt-tooltip text="Apparition rapide" delay="200">
  <span class="tooltip-icon">?</span>
</dt-tooltip>

<!-- Création programmatique -->
const tooltip = DtTooltip.create('Texte du tooltip', {
  position: 'bottom',
  theme: 'success',
  delay: 300
});

Box Components

Le composant <dt-box> est la base pour créer des alertes, toasts et autres composants de notification.

Composant de base (dt-box) :

Ceci est un box de base avec titre et icône. Box de succès avec bouton de fermeture. Box d'avertissement en petite taille. Box d'erreur en grande taille avec titre.

Composant Alert (dt-alert) :

Alerte primaire avec icône automatique. Alerte de succès fermable. Alerte d'avertissement avec icône automatique. Alerte d'erreur avec icône automatique.

Composant Toast (dt-toast) :

Les toasts apparaissent en overlay et se ferment automatiquement :

Toast Succès Toast Avertissement Toast Erreur Toast Info

Code d'exemple :

<!-- Box de base -->
<dt-box variant="primary" icon="info" title="Titre" dismissible>
  Contenu du box
</dt-box>

<!-- Alert dérivé -->
<dt-alert variant="success" dismissible>
  Message de succès
</dt-alert>

<!-- Toast programmatique -->
<script>
DtToast.show('Message de toast', {
  variant: 'success',
  duration: 3000,
  position: 'top-right'
});
</script>

Dark Theme

Le toolkit supporte automatiquement le thème sombre avec plusieurs options :

  • Détection automatique : Respecte les préférences système
  • Basculement manuel : Bouton de thème dans le header
  • Persistance : Sauvegarde le choix dans localStorage
  • Transitions fluides : Animations lors du changement

Classes utilitaires pour le thème :

.bg-theme-primary
.text-theme-primary
.border-theme-primary
.bg-theme-secondary
.text-theme-secondary
.border-theme-secondary
