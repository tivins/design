Les alertes permettent d'afficher des messages importants à l'utilisateur avec différents niveaux de priorité.

Types d'alertes :

Information importante ! Ceci est une alerte primaire avec une icône. Succès ! Votre action a été effectuée avec succès. Attention ! Veuillez vérifier vos informations avant de continuer. Erreur ! Une erreur s'est produite lors du traitement de votre demande. Astuce : Vous pouvez utiliser les raccourcis clavier pour naviguer plus rapidement.

Alertes avec liens :

Cette alerte contient un lien utile pour plus d'informations.

Alertes avec titre :

Vous avez terminé toutes les étapes avec succès. Votre compte est maintenant configuré.

Vous pouvez maintenant commencer à utiliser toutes les fonctionnalités.

Alertes fermables :

Notification : Cette alerte peut être fermée en cliquant sur le bouton X. Nouvelle fonctionnalité : Découvrez notre nouveau système de notifications.

Alertes avec icônes personnalisées :

Fonctionnalité premium : Accédez à des fonctionnalités avancées avec notre plan premium. Merci ! Nous apprécions votre fidélité et votre confiance. Téléchargement disponible : Une nouvelle version est disponible pour téléchargement.

Dark Theme

Le toolkit supporte automatiquement le thème sombre avec plusieurs options :

  • Détection automatique : Respecte les préférences système
  • Basculement manuel : Bouton de thème dans le header
  • Persistance : Sauvegarde le choix dans localStorage
  • Transitions fluides : Animations lors du changement

Classes utilitaires pour le thème :

.text-theme-primary
.border-theme-primary
.bg-theme-secondary
