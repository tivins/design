Theme Toggle

The <dt-theme-toggle> component allows switching between light and dark themes with automatic persistence.

Available Variants:

The component supports different button variants:

Link (default)
Primary
Secondary
Outline Primary
Outline Secondary
Outline Success
Outline Danger
Outline Warning
Outline Info

Available Sizes:

Small (sm)
Medium (md - default)
Large (lg)

Features:

  • Automatic Detection: Respects system preferences on first load
  • Persistence: Saves choice in localStorage
  • Adaptive Icons: Shows moon/sun based on current theme
  • Events: Emits a themeChanged event on change
  • Public API: Methods to control theme programmatically
